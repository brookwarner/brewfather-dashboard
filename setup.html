1 Get Your API Credentials

Go to Brewfather Settings and copy your:

  • API Key
2 Create .env File

In your project root, create a .env file with your credentials:

BREWFATHER_USER_ID=your_user_id_here BREWFATHER_API_KEY=your_api_key_here

npm install npm run dev
